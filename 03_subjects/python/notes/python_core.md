## 핵심 개념

- 변수: 이름에 값을 할당해서 나중에 다시 사용할 수 있게 한다.
- 타입: 값의 종류를 뜻한다. 예: `int`, `str`, `list`, `dict`, `bool`.
- 조건문: `if`로 조건이 참일 때만 특정 코드를 실행한다.
- 반복문: `for`로 리스트 같은 여러 값을 하나씩 꺼내 같은 작업을 반복한다.
- list: 순서가 있는 값 모음이며 인덱스로 접근한다.
- dict: key-value 쌍으로 값을 저장하며, 같은 key에 다시 값을 넣으면 value가 갱신된다.
- function: 반복되는 로직을 이름 붙여 재사용한다. parameter는 정의할 때의 이름, argument는 호출할 때 넘기는 실제 값이다.
- class: 관련 데이터와 그 데이터를 다루는 함수를 하나의 타입으로 묶는 틀이다. Day 1에서는 `dict`보다 구조화된 데이터 표현 방식으로만 이해한다.

## 내가 헷갈린 부분

- `dict`의 중복 제한은 전체 값이 아니라 key에 적용된다.
- Python 3.7+의 `dict`는 삽입 순서를 보존한다.
- parameter와 argument는 반대로 외우기 쉬우므로 구분해서 복습한다.
- 리스트를 `print`하면 요소만 나열되는 것이 아니라 대괄호와 문자열 따옴표를 포함한 리스트 표현으로 출력된다.
- `dict`는 key로 값을 꺼내고, 객체는 보통 점 표기법으로 속성에 접근한다. 예: `student["name"]`, `student.name`.
- `class`는 데이터 자체라기보다 객체를 만들기 위한 설계도다.

## 내 말로 설명

반복문은 여러 데이터를 하나씩 꺼내 처리하는 구조다. 조건문을 반복문 안에 넣으면 원하는 데이터만 골라낼 수 있다. `dict`는 이름이나 점수처럼 관련된 값을 key로 꺼낼 때 유용하다. 함수는 이런 처리 과정을 이름 붙여 다시 사용할 수 있게 한다.
