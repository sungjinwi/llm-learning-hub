## 사용 규칙

1. 사용자의 질문을 `04_ai_prompts/method_router.md`로 분류한다.
2. 해당 Day 또는 주제의 실습 파일을 연다.
3. 정답을 먼저 보여주지 않는다.
4. 사용자가 답하거나 코드를 작성하면 채점한다.
5. 틀린 내용은 `06_tracking/weak_points.md`에 추가할 형식으로 정리한다.
