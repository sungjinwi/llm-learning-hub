# Master Plan

보고서의 1주 집중 커리큘럼을 로컬 학습 운영 계획으로 정리한다.

## 전체 목표

7일 동안 Python 기초에서 시작해 데이터 처리, 머신러닝 파이프라인, PyTorch 딥러닝 훈련 루프, 엔드투엔드 미니 프로젝트까지 경험한다.

목표는 완벽한 숙달이 아니라 다음 네 가지다.

1. 핵심 개념의 뼈대 만들기
2. AI를 튜터와 검증 도구로 사용하는 습관 만들기
3. 능동적 회상과 오답 기록으로 장기 기억 기반 만들기
4. AI가 만든 코드도 스스로 해석하고 고칠 수 있게 만들기

## 7일 커리큘럼

| Day | 주제 | 핵심 산출물 | 기본 학습법 |
| --- | --- | --- | --- |
| 1 | Python 기본 문법과 자료구조 | 변수, 조건문, 반복문, list, dict, function 노트 | 소크라테스 문답, 능동적 회상 |
| 2 | NumPy와 Pandas | 배열, 브로드캐스팅, DataFrame 조작 실습 | 단계별 출력 추적 |
| 3 | EDA와 전처리 | 결측치, 이상치, 정규화, 인코딩 실습 | 오류 데이터 복구 훈련 |
| 4 | Scikit-Learn 머신러닝 | train/test split, cross validation, Pipeline | 역공학, 평가 지표 회상 |
| 5 | PyTorch Tensor와 Autograd | tensor shape, device, computational graph | 코드와 수식 1:1 매핑 |
| 6 | 신경망과 Training Loop | nn.Module, loss, optimizer, 5단계 루프 | 파인만 설명, 빈칸 코드 |
| 7 | 엔드투엔드 프로젝트 | 데이터 로딩부터 모델 저장까지 하나의 파이프라인 | 코드 리뷰, 리팩토링 |

## 매일 완료 기준

하루 학습은 다음 조건을 만족해야 완료로 본다.

- 핵심 개념 노트 1개 이상 작성
- 능동적 회상 문제 5개 이상 풀이
- 틀린 문제 또는 헷갈린 개념을 `06_tracking/weak_points.md`에 기록
- 다음 복습 항목을 `06_tracking/review_schedule.md`에 기록
- 오늘 배운 내용을 5문장 이내로 파인만식 설명
