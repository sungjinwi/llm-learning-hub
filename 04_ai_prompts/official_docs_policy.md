# Official Docs Policy

정확도가 중요한 학습 주제는 공식 문서를 우선한다.

## 공식 문서가 필요한 주제

- OpenAI API, 모델, embeddings, structured output, tool calling
- LangChain, LangGraph, Gradio, Hugging Face
- Pandas, NumPy, scikit-learn
- Python 표준 라이브러리 중 버전 차이가 있는 기능
- API key, rate limit, 비용, 배포, 보안
- accuracy, precision, recall, F1, 검색 평가 지표처럼 오해 비용이 큰 평가 개념

## 공식 문서를 생략할 수 있는 주제

- 기초 개념 복습
- 출력 예측 문제
- 사용자의 약점 재출제
- 이미 로컬 노트에서 다룬 개념의 확인 문제

## 답변 원칙

1. 문서 기준 사실과 AI의 해석을 구분한다.
2. 문서를 읽은 뒤에도 바로 정답을 주지 말고, 가능한 한 작은 실습으로 연결한다.
3. 공식 문서가 여러 버전으로 갈리면 버전 또는 날짜를 명시한다.
4. 문서로 확인하지 않은 최신 정보는 확정적으로 말하지 않는다.
5. 학습 로그에는 참고한 공식 문서와 확인한 항목을 짧게 남긴다.
