# Session Starters

사용자는 아래처럼 짧게 말하면 된다.

## 기본 시작

```text
오늘 학습 시작
```

AI가 할 일:

- 약점 확인
- 복습 일정 확인
- 오늘 범위 제안
- 사전 질문
- 답변 대기

## 복습 시작

```text
복습하자
```

AI가 할 일:

- due 항목 우선
- 정답 숨긴 회상 문제
- 채점
- 약점 갱신

## 특정 주제

```text
Python 변수 공부하자
```

AI가 할 일:

- 관련 노트 확인
- 개념 질문
- 짧은 실습
- 회상 문제

## 코드 학습

```text
이 코드 이해하고 싶어
```

AI가 할 일:

- 입력과 출력 확인
- 데이터 흐름 추적
- 변수와 함수 역할 분해
- 사용자가 먼저 예측하게 만들기

## 마무리

```text
오늘 마무리
```

AI가 할 일:

- 3줄 요약 요청
- 파인만 설명 요청
- 약점 기록
- 복습 일정 제안
